Краткое описание ПО

IPR РКИ

Специализированная образовательная платформа, полностью адаптированная для иностранных студентов, изучающих русский язык и интересующихся русской культурой

(Программное обеспечение «Русский как иностранный» с интегрированной базой данных «РУССКИЙ КАК ИНОСТРАННЫЙ»)

Возможности платформы

Функциональные характеристики

ПО представляет собой интернет-сервис, дополнительных требований по установке программы со стороны пользователя не требуется

  • Размещение контента собственного бренда и ведущих издательств учебной и научной литературы
  • Расширенный каталог с возможностью расширенного поиска для более точных результатов
  • Пакетная регистрация читателей (добавление пользователей списком)
  • Личный кабинет читателя с функционалом цитирования, закладок
  • Интеграция с LMS Moodle, API
  • Подключение к ЭИОС учебных заведений (бесшовный переход)
  • Сервис каталогизатора книг для удобного управления фондами
  • Экспорт MARC-записей с поддержкой новых форматов и кодировок
  • Сбор и просмотр статистики цифровой активности пользователей
  • Детальная аналитика и статистика активности пользователей
  • Формирование отчётов на основе данных, которые есть у платформы
  • Взаимодействие библиотеки с читателями (заявки, детализация данных)
  • Полнофункциональная версия платформы для слабовидящих пользователей

Технологии

Технологический стек продукта

При разработке ПО «Русский как иностранный» с интегрированной базой данных «РУССКИЙ КАК ИНОСТРАННЫЙ» используются следующие языки программирования, программное обеспечение, технологии и инструменты разработки:

  • Языки программирования: PHP, Python, JavaScript / TypeScript, SQL;
  • Фреймворки и библиотеки: Laravel, HTML, CSS, Bootstrap;
  • Базы данных и поисковые системы: MariaDB, Solr, Memcached;
  • Инфраструктура и DevOps: Git, REST API, Swagger/OpenAPI.
